IU-李知恩

李知恩[b]（朝鲜语：이지은 Lee Ji Eun，英语：Lee Ji Eun，1993年5月16日－），艺名IU（朝鲜语：아이유 A I Yu ），韩国创作歌手、女演员。

个人介绍

       她于2007年与LOEN娱乐签约成为练习生，15岁时推出首张迷你专辑《Lost and Found》，开始歌手生涯。之后发行的专辑《Growing Up》和《IU...IM》使IU的知名度上涨。真正让IU在韩国全国走红的是她在2010年发布的专辑《Real》的主打歌《好日子》（韩语：좋은 날）。《好日子》连续五周位居韩国Gaon数位榜榜首，与PSY的《江南Style》共同保持着维持一位周数最长的记录。在2011年发行的专辑《Real+》和《Last Fantasy》相继取得成功后，IU在韩国的音乐排行榜确立强势地位，并进一步巩固邻家女孩的形象，得到“国民妹妹”的称号。IU在2013年发行的第3张正规专辑《Modern Times》展现成熟风格，与之前树立的少女形象不同。IU后来发行的专辑《花书签》和《CHAT-SHIRE》都与韩国的主流音乐风格背离，却没有动摇她在音乐排行榜的主导地位。 IU共发布4张录音室专辑和9张迷你专辑，在Gaon榜有2张专辑和14首歌曲获得第一名。在男女音乐团体团体为主的韩国流行音乐产业中，IU 是最畅销的独唱艺人之一。自2012年起一直位列《福布斯》杂志的年度韩国名人权力榜40强之一，2012年达到最高的第3名。告示牌将IU认可为韩国流行音乐百强榜的先锋，她是拥有最多的冠军歌曲并且是维持第一位置周数最长的艺人。

早年生活

IU于1993年5月16日生于韩国，自小在首尔长大，与其父母和弟弟住在一起。IU上小学时，家庭经济状况恶化，父母因为为他人作担保而欠下了巨债，一家人被逼要分开生活，她和弟弟被迫与父母分开居住，与IU的奶奶和表姐在一间公寓中度过了一年多贫困的生活。这一段时间，IU与她的父母接触不多，因而很感激奶奶的出现。 在上中学时，IU发现了她对唱歌的热情。在校内运动会演唱并赢得了老师及同学的掌声后，IU萌生了成为歌手的想法。她曾先后参加过20次选拔但均被拒绝，并且曾被伪造的娱乐公司欺骗。2007年10月，IU通过现时经纪公司LOEN娱乐的试镜，并成为练习生。尽管认为经过三到四年的训练后她可能最终会被选入一个女子团体，但她在训练10个月后便随即发行她的首张个人专辑。因为曾经艰苦的生活，IU说她“喜欢在练习室”，在那里她可以吃她想吃的东西，并且有睡觉的地方。在出道之前，LOEN娱乐将其艺名命名为“IU”。艺名从“我和你”（I and You）中派生而出，象征着音乐在人与人之间所产生的凝聚力。 IU歌唱事业的迅速发展导致她时常旷课，学习成绩除韩国文学外均有所下降。在2011年IU于同德女子高中毕业后，IU决定继续她的歌唱事业，放弃进一步的高等教育。

公益

2011－2015年

2011年，IU举行了第一次粉丝见面会“好日子，因为和IU在一起”，所得收入悉数捐给贫困和弱势民众。同年，她捐出与韩国“冰上精灵”金妍儿合唱曲《冰花》（Ice Flower）的收入，另外与歌手边镇燮、金泰源一同举办的演唱会收入同样捐作慈善。2013年起，IU连续5年每年捐款1,000万韩元给母校同德女子高中，提供2,000万韩元奖学金给考上大学的低收入家庭的学生。 2014年，韩国发生世越号沉船事故，IU将当年的演唱会收入全部捐给罹难者。2015年，IU又为儿童福利机构捐款1亿韩元，帮助低收入家庭的儿童，为经济困难的单亲家庭等提供经济支援。在贫穷中长大的IU一直致力于慈善公益事业，在以上五年中，她每年向各种慈善机构捐款约3亿至5亿韩元。

2016－2020年

2016年起，IU每年都会为瑞草区捐献1000万韩元，今年的捐款将用于说明独居老人等弱势群体度过酷暑。瑞草区政府工作人员透露，IU在捐献这笔钱时特意表示不希望捐献一事被公之于众，但因为传达过程中政府工作人员的失误，导致IU捐款的事情被公之于众。 2018年9月18日IU迎来出道10周年之际，还与“UAENA（官方粉丝名）”共同捐款1亿韩圜帮助祖孙家庭及低收入青少年，提供弱势族群奖学金。 2019年3月，IU在出演后戏剧《我的大叔》后受到角色的启发，她向“韩国聋人协会”捐款帮助听障和言语障碍人士。同年4月，韩国江原道的森林大火导致近4200人无家可归，IU捐款1亿万韩圜帮助江原道森林火灾后无家可归的近4,200人，她是首位捐款以帮助救灾工作的K-POP艺人。同时为了对江原道火灾的受害者表示尊重，原预计于4月5日上映由IU主演的微电影作品《女孩，四绎》也因为剧中的森林火灾画面而推迟至4月11日发布。同年5月，在韩国儿童节来临之际她向机构“Childcare Korea”捐赠了1亿韩元，为发展迟缓的小孩和贫困家庭儿童给予协助。 同年9月，IU更以粉丝 Uaena 的名义捐款一亿韩元，来庆祝自己出道满十一周年。同年12月2日，福布斯官网公开发表了一篇名为《2019年亚洲慈善英雄：变革的催化剂》的文章，IU被列入为其中一名“2019亚洲慈善英雄”是本次最年轻的入选者。文中提到IU自2018年以来已经向各类机构和事业捐赠了9亿韩元（约港币589万元），表扬她对慈善工作不遗余力，善举亦值得被认可。 2020年2月26日， IU为新型冠状病毒肺炎疫情事件严重的大邱地区，慷慨捐助了2亿韩元（约16万5千美金）购买防疫物资予以低收入户、弱势社群和一线医疗人员她表示：“听到消息说医疗机关的医护人员们穿着的防护服非常匮乏，希望为了给守护国民生命与安全，支援在尽力为患者治疗的医护人员们，哪怕只是微不足道的帮助，因此进行了捐赠。” 2020年3月1日，IU捐赠3000万韩元予以自己居住的首尔瑞草区低收入户和单亲家庭用于购买防疫物资，应付严峻的新型冠状病毒肺炎疫情。3月2日IU通过家人转达，为预防京畿道杨平郡内弱势群体的新型冠状病毒肺炎，捐赠了2000万韩元。3月3日，IU再次捐赠3000万给果川市，用于预防新型冠状病毒肺炎疫情之用 。IU曾在2018于果川市购买不动产作为个人和母亲的办公室、仓库和后辈使用的练习室，并在2019年为果川市的低收入户捐款1千万韩元。果川市政府计划用收到的捐款购买口罩和手消毒剂等防疫物品，转交给辖区内的设施和弱势群体。3月5日，IU再次向杨平郡的福利机构捐赠了2000万韩元，同时对儿童保护机构、重症残疾人福利机构等共计捐款1500万韩元。截至目前为止，IU在这次疫情中总共捐赠了3.15亿韩元（约26.6万美金）。

仙女照片

音乐作品

IU共发行4张正规专辑、5张迷你专辑（不包含改版迷你专辑）、35首单曲及37部MV。除此之外，她还发行了12首合作歌曲、17首电视剧、电影和网络游戏的原声音乐（包含未正式发行的原声音乐），她亦为其他歌手填词作曲。 在Gaon专辑榜和Gaon音源榜，IU分别有2张专辑和14首歌曲获得一位。她还6次登上美国Billboard的韩国流行音乐百强榜（Korea K-Pop Hot 100，已停止排行），成为榜单中获得一位歌曲最多的歌手。自2008年出道以来，IU在韩国专辑销量达45万张，数字音乐下载量7000万次（包含专辑歌曲、合作歌曲、原声音乐及其他参与歌曲）。
